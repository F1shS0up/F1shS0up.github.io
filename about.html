Hi my name is

Adam Vlčko

Hi my name is Adam Vlčko. I am game developer, programmer, artist and game designer from Czechia. I specialize in:

  • Programming languages: C++, C#, HTML, Python
  • Game engines: Unity, Unreal Engine(4, 5)
  • Frameworks: XNA(Monogame), SDL
  • Other stuff: OpenGL

Career

2017

I started programming in C++. I was playing games a lot and I wondered how are they made. That idea brought me to C++. I used borrowed laptop for it, it wasn't anything fast but it did the job. I learned some basics like variables, functions, struct, classes... I made few projects in console like crossy road, snake or tetris.

2018

In 2018 I started thinking more about game development rather than programming. I learned that there are so called "Game engines" that make creating games easier and fast. I tried to use Unreal Engine, because only language I knew was C++. However unfortunately the performance was really bad, so in summer I got new laptop. I downloaded Unreal Engine 4. I started learning some basics but after a while I started realizing that 3D games may be too tough to handle...

2019

I continued with Unreal Engine but after a while I decided to switch to Unity, because I wanted to do 2D games. I learned C# pretty quickly, after all it's all object oriented programming. With unity I felt confident so after 2 months, I decided to make a quick game. I didn't publish it anywhere yet because I didn't have any idea where.

2020

In this year I didn't do much just few projects...

2021

My itch.io page was made and I made my discord community. I released the game from 2019 and I call it Grav. I started working more and more and after few failed projects, I thought of making mobile game. After almost half a year of work I published it on google play (unfortunately my acount got blocked) and I called it Dunk IT!

2022

In the first half I participated in my first game jam. It was called Black And White Jam. I finished 46th out of 341 entries. It wasn't the best result but it was atleast something. The game was called Mehtris. I got bored of Unity but back then I liked C# more so I got the idea of using framework. At that time I played stardew valley a lot so I decided to make a game in Monogame(XNA). I liked it a lot, therefore I participated in another Game Jam called The GameDev.tv Game Jam. I made game called Suffering and it ended up finishing 88th out of 933 entries. Again nothing to be really happy about but it wasn't that bad result. In the second half I thought of making a bigger game but it didn't work out well.

2023

I joined Team Melon and we did one game jam together. We ended up placing 3rd which was an awesome result. Even tho I am a programmer I was the main artist for this game. The game is called Cottage ville. We continued to develop this game and we ended up putting it on steam half a year later. We also did monkey jam where we finished 11th. We called the game for monkey jam ESC-Ape. This year I was also working on a game for an art competition, that has been hosted in my country. We finished 3rd, which absolutely blew my mind, because the competition is known across whole Czechia. The game I am talking about is Fairtrade Ride. I got a part time job, where I saved enough money to get a really decent computer. After that I started working on my 2D Game Engine.
