How I made my website

Why?

First, I want to talk about why I decided to make a website. I started thinking about making a website when I needed a place for a portfolio and to keep my games and projects together. I thought itch.io would be enough, but unfortunately, I wanted a completely different page for blogs and devlogs, which itch.io doesn't yet have. I also wanted to practice HTML and CSS.

Starting point

I started by searching for website builders like Wix and Monokeep, but they weren't free. I messaged my friend and co-worker Dan, who has already made his website. He said that I should check out HTML templates. When I was searching for a template, I found a really great website called templated.co where I found a template with the name HIELO. This was my starting point for making this website.

Main development

In this stage, I started with the index (home page). I changed some graphics, created some links, rewrote some paragraphs, changed my social media, and drew some art. It wasn't that hard, but organizing and realizing how it works was a bit tough. After that, I created a few new pages: About, Blogs, and Game Engine. I also made Blog0, which is this blog you're reading right now. I skipped the About page because the About page is basically about my career, and it is just text. On the Game Engine page, I created a banner and some titles; I didn't do anything else because again, it would probably be just text and maybe a few links. I moved to probably the most complex one, and that was the blogs page.

Blogs

First, I made the banner, and then I copied the grid layout from the index and put it in blogs. I created my first blog called, as you guessed it, "How I Made My Website". I found some free-to-use website images, edited them a bit, and then used them. I found out that at one resolution, the headline of the page ("How I Made My Website") is glitching when the resolution is too low. To fix it, I had to change the font size in CSS. In detail, I had to change the units from em to vw so that it dynamically scaled. That seemed to work. This is most of the work, the only thing left is the text...
